## User Control

You control whether switchboard modes, Headroom routing, RTK integration, and
add-ons are enabled. Off mode is intended to stop managed local proxy routing
and disable managed RTK integration until you turn those features back on.

The app may edit local configuration files only for features you enable. Managed
edits should be reversible, backed up where applicable, and visible through app
diagnostics.

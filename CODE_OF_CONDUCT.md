# Code of Conduct

Mac AI Switchboard welcomes useful, respectful contributions.

## Expected Behavior

- Be clear, patient, and constructive.
- Keep technical disagreements focused on the code, docs, evidence, and user
  safety.
- Respect maintainer decisions about product direction, release readiness,
  security, privacy, and branding.
- Report security issues privately under `SECURITY.md`.
- Do not pressure maintainers to merge unreviewed or unwanted work.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing another person's private information.
- Posting secrets, credentials, or private keys.
- Repeatedly reopening rejected proposals without new evidence.
- Misrepresenting a fork, build, or release as official.
- Attempting to bypass maintainer approval or branch protection.

## Enforcement

The maintainer may close issues, reject pull requests, delete comments, block
users, or report abuse to GitHub when needed to protect the project and its
users.

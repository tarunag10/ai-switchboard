# Support

Mac AI Switchboard is a free open-source project.

## Community Support

Use GitHub Issues for reproducible bugs, documentation gaps, and narrowly scoped
feature requests.

Before opening an issue:

- Search existing issues.
- Confirm you are using a current commit or release.
- Remove secrets from logs and screenshots.
- Include macOS version, chip architecture, app version, and switchboard mode.

## Security Reports

Do not open a public issue for security vulnerabilities. Follow `SECURITY.md`.

## Official Support Boundary

Official support applies only to this repository and official maintainer-approved
builds. Forks and redistributed builds should provide their own support channel,
app name, bundle identifier, signing identity, and update channel.

## No Paid Account Requirement

The public app is intended to be free and local-first. It should not require
sign-in, checkout, pricing APIs, or paid Mac AI Switchboard services to run.
